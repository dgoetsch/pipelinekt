Node(label: Str, steps: Step)

You should not have to use this step. This is from the old scripted pipelines. However, this is present for users who require this functionality; for example parallelism that exceeds the max allowed by declarative stages, see com.code42.jenkins.pipelinekt.step.scripted.ScriptedParallel

Using this method causes the Blue Ocean UI to misbhave - the ui gets slow and stages don't display correctly.
